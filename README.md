# bb-creation-diy

학생에게 다음과 같은 절차로 벽돌깨기 게임을 만드는 방법을 따라하기 형식으로 가르쳐줄수 있는 매뉴얼을 필요한 웹링크(예 firebase, github, claude 등)가 포함된 html 파일로 작성해줘. 

게임 생성 절차

1. Claude, LMArena에서 게임 html 페이지 생성 (자세히 만들고 싶은 게임 만들어달라고 하면 됨, 단 사용자가 이름을 입력하고 로그인하게 한 뒤 게임결과점수 및 순위관리를 할수 있도록 firebase DB연동해달라고 요청)

2. 구글 파이어베이스 실시간 DB생성 및 APIKey 복사

3. Github에서 게임 퍼블리쉬 
    단계1: 레파지토리 생성 (프로젝트 개념)
    단계2: index.html upload (업로드 후 반드시 commit 버튼 클릭)
    단계3: 설정/페이지/ 브랜치(가지) -> 기본으로 설정 후 저장

4. 상단의 프로젝트명 클릭 후 30초간 대기 (브라운색깔 점이 -> 연두색 체크로 변환됨)

5. 설정/페이지 --> 사이트 주소 클릭하면 게임사이트로 이동됨

제출은 사이트 주소만 입력하면 됨
학생들이 따라하기 쉬운 벽돌깨기 게임 제작 매뉴얼 HTML 파일을 작성해드리겠습니다.
